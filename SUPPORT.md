# Support

## Community (this repository)

- Channel: GitHub Issues.
- Model: best effort.
- No response/correction SLA.

## Enterprise

- Dedicated support.
- Contractual SLA.
- Channel and response times defined in the contract.

## Best practices when opening an issue

Include:
- context and expected behavior;
- relevant logs (without secrets);
- version/commit;
- steps to reproduce.
